# 增长检查清单

P3 阶段 Growth Lead / Marketing Strategist 使用。

## 必检项

- [ ] **自然流量占比** — 非纯依赖付费投放
- [ ] **SEO 基础** — 关键词布局、技术 SEO、内容资产
- [ ] **GEO 可见性** — 能被 LLM 引用（结构化数据、权威信号）
- [ ] **留存数据** — D1/D7/D30 有监测，有流失节点分析
- [ ] **裂变机制** — K 因子可计算，传播路径存在
- [ ] **品牌资产** — 视觉/调性一致，有心智占位
- [ ] **内容生产** — 可持续的内容产出能力
- [ ] **获客渠道多元化** — ≥2 个独立获客渠道
- [ ] **商业闭环** — 获客→激活→留存→变现→推荐链路完整
- [ ] **数据驱动** — 有漏斗监测、A/B 测试能力

## 红线

- [ ] 只看获客不看留存（漏水桶）
- [ ] 无留存数据的增长计划
- [ ] 增长依赖单一付费渠道且 CAC 上升
- [ ] 把"能做增长"当"已经在增长"
- [ ] 无品牌沉淀，纯流量生意
