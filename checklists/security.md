# 安全检查清单

P3 阶段 Security Auditor / Compliance Officer 使用。

## 必检项

- [ ] **认证机制** — 密码策略、MFA、会话超时、token 轮换
- [ ] **授权粒度** — 最小权限，无越权，IDOR 防护
- [ ] **输入验证** — 所有外部输入经校验（注入防护）
- [ ] **输出编码** — XSS 防护，上下文感知编码
- [ ] **数据加密** — 传输(TLS) + 存储(敏感字段)，密钥非硬编码
- [ ] **PII 处理** — 最小收集、脱敏、用户知情同意
- [ ] **API 安全** — 限流、签名、防重放
- [ ] **依赖安全** — SBOM、漏洞扫描、License 合规
- [ ] **日志脱敏** — 敏感数据不入日志
- [ ] **审计日志** — 关键操作可追溯

## 红线

- [ ] 硬编码密钥/token/密码
- [ ] 明文存储密码或敏感数据
- [ ] 信任客户端鉴权
- [ ] 无限流的公开 API
- [ ] 隐私政策与实际行为不符
- [ ] License 不兼容的开源依赖（如 GPL 进闭源商业产品）
