# 商业检查清单

P3 阶段 VC / Sales / Growth 使用。

## 必检项

- [ ] **商业模式清晰** — 谁付钱、付什么钱、为什么付
- [ ] **单位经济模型** — LTV / CAC 可计算且 LTV/CAC ≥3
- [ ] **收入可持续性** — 不是一次性收入，有续费/扩展路径
- [ ] **市场时机** — 窗口期打开，非红海晚期
- [ ] **增长飞轮** — 自循环机制成立（不靠无限烧钱）
- [ ] **退出路径** — IPO / 并购的潜在买家或路径
- [ ] **定价策略** — 与价值匹配，有提价空间
- [ ] **销售周期** — 决策链长度可控，对现金流压力可承受
- [ ] **NRR ≥100%** — 现有客户不流失且在扩展
- [ ] **渠道多元化** — 不依赖单一获客渠道

## 红线

- [ ] CAC > LTV 且无改善路径
- [ ] 无续费数据的订阅模式
- [ ] 单一客户贡献 >50% 收入
- [ ] "大市场"替代"我们能拿多少"
